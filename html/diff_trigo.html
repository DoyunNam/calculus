Differentiation of Trigonometric Functions

The differentiation of the six basic trigonometric functions are as follows:

  • \(\frac{d}{dx} \sin(x) = \cos(x)\)
  • \(\frac{d}{dx} \cos(x) = -\sin(x)\)
  • \(\frac{d}{dx} \tan(x) = \sec^2(x)\)
  • \(\frac{d}{dx} \cot(x) = -\csc^2(x)\)
  • \(\frac{d}{dx} \sec(x) = \sec(x) \tan(x)\)
  • \(\frac{d}{dx} \csc(x) = -\csc(x) \cot(x)\)

Proofs

1. \(\frac{d}{dx} \sin(x) = \cos(x)\)

Using the limit definition of the derivative:

\[ \frac{d}{dx} \sin(x) = \lim_{{h \to 0}} \frac{\sin(x+h) - \sin(x)}{h} \]

Using the trigonometric identity for \(\sin(a+b)\):

\[ \sin(x+h) = \sin(x)\cos(h) + \cos(x)\sin(h) \]

Substituting this into the limit expression:

\[ \frac{d}{dx} \sin(x) = \lim_{{h \to 0}} \frac{\sin(x)\cos(h) + \cos(x)\sin(h) - \sin(x)}{h} \]

Rearranging terms:

\[ \frac{d}{dx} \sin(x) = \lim_{{h \to 0}} \left( \sin(x) \frac{\cos(h) - 1}{h} + \cos(x) \frac{\sin(h)}{h} \right) \]

Using the known limits \(\lim_{{h \to 0}} \frac{\sin(h)}{h} = 1\) and \(\lim_{{h \to 0}} \frac{\cos(h) - 1}{h} = 0\):

\[ \frac{d}{dx} \sin(x) = \sin(x) \cdot 0 + \cos(x) \cdot 1 = \cos(x) \]

2. \(\frac{d}{dx} \cos(x) = -\sin(x)\)

Using the limit definition of the derivative:

\[ \frac{d}{dx} \cos(x) = \lim_{{h \to 0}} \frac{\cos(x+h) - \cos(x)}{h} \]

Using the trigonometric identity for \(\cos(a+b)\):

\[ \cos(x+h) = \cos(x)\cos(h) - \sin(x)\sin(h) \]

Substituting this into the limit expression:

\[ \frac{d}{dx} \cos(x) = \lim_{{h \to 0}} \frac{\cos(x)\cos(h) - \sin(x)\sin(h) - \cos(x)}{h} \]

Rearranging terms:

\[ \frac{d}{dx} \cos(x) = \lim_{{h \to 0}} \left( \cos(x) \frac{\cos(h) - 1}{h} - \sin(x) \frac{\sin(h)}{h} \right) \]

Using the known limits \(\lim_{{h \to 0}} \frac{\sin(h)}{h} = 1\) and \(\lim_{{h \to 0}} \frac{\cos(h) - 1}{h} = 0\):

\[ \frac{d}{dx} \cos(x) = \cos(x) \cdot 0 - \sin(x) \cdot 1 = -\sin(x) \]

3. \(\frac{d}{dx} \tan(x) = \sec^2(x)\)

Using the quotient rule \(\frac{d}{dx} \left(\frac{u}{v}\right) = \frac{u'v - uv'}{v^2}\) where \(u = \sin(x)\) and \(v = \cos(x)\):

\[ \frac{d}{dx} \tan(x) = \frac{d}{dx} \left(\frac{\sin(x)}{\cos(x)}\right) = \frac{\cos(x)\cos(x) - (-\sin(x))\sin(x)}{\cos^2(x)} \]

Simplifying the numerator:

\[ = \frac{\cos^2(x) + \sin^2(x)}{\cos^2(x)} = \frac{1}{\cos^2(x)} = \sec^2(x) \]

4. \(\frac{d}{dx} \cot(x) = -\csc^2(x)\)

Using the quotient rule where \(u = \cos(x)\) and \(v = \sin(x)\):

\[ \frac{d}{dx} \cot(x) = \frac{d}{dx} \left(\frac{\cos(x)}{\sin(x)}\right) = \frac{-\sin(x)\sin(x) - \cos(x)\cos(x)}{\sin^2(x)} \]

Simplifying the numerator:

\[ = \frac{-\sin^2(x) - \cos^2(x)}{\sin^2(x)} = \frac{-1}{\sin^2(x)} = -\csc^2(x) \]

5. \(\frac{d}{dx} \sec(x) = \sec(x) \tan(x)\)

Using the quotient rule where \(u = 1\) and \(v = \cos(x)\):

\[ \frac{d}{dx} \sec(x) = \frac{d}{dx} \left(\frac{1}{\cos(x)}\right) = \frac{0 \cdot \cos(x) - 1 \cdot (-\sin(x))}{\cos^2(x)} \]

Simplifying:

\[ = \frac{\sin(x)}{\cos^2(x)} = \frac{1}{\cos(x)} \cdot \frac{\sin(x)}{\cos(x)} = \sec(x) \tan(x) \]

6. \(\frac{d}{dx} \csc(x) = -\csc(x) \cot(x)\)

Using the quotient rule where \(u = 1\) and \(v = \sin(x)\):

\[ \frac{d}{dx} \csc(x) = \frac{d}{dx} \left(\frac{1}{\sin(x)}\right) = \frac{0 \cdot \sin(x) - 1 \cdot \cos(x)}{\sin^2(x)} \]

Simplifying:

\[ = \frac{-\cos(x)}{\sin^2(x)} = -\frac{1}{\sin(x)} \cdot \frac{\cos(x)}{\sin(x)} = -\csc(x) \cot(x) \]
