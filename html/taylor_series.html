Taylor Series

The Taylor series of a function \( f(x) \) about \( x = a \) is given by:

\[ f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x - a)^n \]

Proof

To derive the Taylor series, we start with the function \( f(x) \) and assume it can be expressed as a power series:

\[ f(x) = \sum_{n=0}^{\infty} c_n (x - a)^n \]

Our goal is to determine the coefficients \( c_n \). By differentiating the series term by term and evaluating at \( x = a \), we obtain:

\[ f(a) = c_0 \] \[ f'(x) = \sum_{n=1}^{\infty} n c_n (x - a)^{n-1} \] \[ f'(a) = c_1 \]

Continuing this process, we find:

\[ f^{(n)}(x) = \sum_{k=n}^{\infty} \frac{k!}{(k-n)!} c_k (x - a)^{k-n} \] \[ f^{(n)}(a) = n! c_n \]

Therefore, the coefficients \( c_n \) are given by:

\[ c_n = \frac{f^{(n)}(a)}{n!} \]

Substituting these back into the power series, we obtain the Taylor series:

\[ f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x - a)^n \]

Examples

Example 1: Exponential Function

The Taylor series for \( e^x \) about \( x = 0 \) is:

\[ e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} \]

Example 2: Sine Function

The Taylor series for \( \sin(x) \) about \( x = 0 \) is:

\[ \sin(x) = \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n+1)!} x^{2n+1} \]

Example 3: Cosine Function

The Taylor series for \( \cos(x) \) about \( x = 0 \) is:

\[ \cos(x) = \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n)!} x^{2n} \]
