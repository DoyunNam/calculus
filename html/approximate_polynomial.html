Approximate Polynomial and Error

Approximate Polynomial

Consider a function \( f(x) \) that is infinitely differentiable in some interval around \( x = a \). We can approximate \( f(x) \) using a polynomial known as the Taylor series:

\[ P_n(x) = f(a) + f'(a)(x - a) + \frac{f''(a)}{2!}(x - a)^2 + \cdots + \frac{f^{(n)}(a)}{n!}(x - a)^n \]

Here, \( P_n(x) \) is the Taylor polynomial of degree \( n \) centered at \( x = a \).

Error in Approximation

The error of approximating \( f(x) \) by \( P_n(x) \) is given by the remainder term \( R_n(x) \):

\[ R_n(x) = f(x) - P_n(x) \]

Using the Lagrange form of the remainder, we have:

\[ R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} (x - a)^{n+1} \]

for some \( \xi \) between \( a \) and \( x \).

Proof

To prove the formula for the remainder term, we use the Mean Value Theorem for integrals and repeated differentiation.

  1. Define a new function:

    \[ g(t) = f(t) - \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!} (t - a)^k \]

    This function \( g(t) \) represents the difference between \( f(t) \) and its \( n \)-th degree Taylor polynomial at \( t = a \).

  2. Note that \( g(a) = 0, g'(a) = 0, \ldots, g^{(n)}(a) = 0 \).
  3. Apply the Mean Value Theorem to \( g^{(n+1)}(t) \) over the interval \([a, x]\):

    \[ g^{(n+1)}(\xi) = \frac{g^{(n)}(x) - g^{(n)}(a)}{x - a} \]

    Given \( g^{(n)}(a) = 0 \), we have:

    \[ g^{(n+1)}(\xi) = \frac{g^{(n)}(x)}{x - a} \]

  4. Repeating this process, we find that \( g(x) \) can be expressed as:

    \[ g(x) = \frac{g^{(n+1)}(\xi)}{(n+1)!} (x - a)^{n+1} \]

  5. Thus, the remainder \( R_n(x) = g(x) \), giving us:

    \[ R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!} (x - a)^{n+1} \]

Examples

  1. Approximation of \( e^x \) near \( x = 0 \):

    \[ e^x \approx 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} \]

    The error term for the 3rd degree polynomial is:

    \[ R_3(x) = \frac{e^\xi}{4!} x^4 \quad \text{for some } \xi \text{ between 0 and } x \]

  2. Approximation of \( \sin(x) \) near \( x = 0 \):

    \[ \sin(x) \approx x - \frac{x^3}{3!} \]

    The error term for the 3rd degree polynomial is:

    \[ R_3(x) = \frac{\cos(\xi)}{4!} x^4 \quad \text{for some } \xi \text{ between 0 and } x \]
